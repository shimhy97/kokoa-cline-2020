Cascading Style Sheet

CSS는 HTML과 같이 써야 한다. 절대 따로 사용하지 않는다.

HTML은 브라우저에게 웹사이트의 컨텐츠에 대해 알려줘.
CSS는 브라우저에게 웹사이트가 어떻게 보여야 하는 지 알려줘. 

ex. HTML : 헤이 브라우저! 이건 타이틀이야! CSS: 헤이! 그 타이틀은 녹색이어야 해! 가로 25px 가로 100px이어야 한다고!
ex. 글씨 위에 커서 올리니 색상이 변한다? --> CSS
ex. 타이틀은 가운데 있어야 하고, 링크는 노랑색이어야 하고... 이 이미지의 크기는..--> CSS
HTML : 뼈대, CSS : 근육 장기
